CSS Cheatsheet

1. Basic Syntax

selector {
    property: value;
}

Example:

h1 {
    color: blue;
}

2. Box Model

The CSS box model consists of margins, borders, padding, and the actual content area.

/* Box Model Example */
.box {
    width: 200px;
    padding: 20px;
    border: 5px solid black;
    margin: 10px;
}

3. Positioning

Types of positioning:

  • static: Default position, follows the normal flow.
  • relative: Positioned relative to its normal position.
  • absolute: Positioned relative to the nearest positioned ancestor.
  • fixed: Positioned relative to the viewport, remains in place when scrolling.
  • sticky: Switches between relative and fixed, depending on scroll position.
.relative {
    position: relative;
    top: 10px;
}

.absolute {
    position: absolute;
    top: 10px;
    left: 10px;
}

4. Display Properties

Common values:

  • block: Takes full width, starts on a new line.
  • inline: Takes only the width needed, does not start on a new line.
  • inline-block: Behaves like inline but allows height and width.
  • flex: Enables flexbox layout.
  • grid: Enables grid layout.
  • none: Hides the element.
.block {
    display: block;
}

.inline {
    display: inline;
}

5. Flexbox

Flexbox properties:

.container {
    display: flex;
    justify-content: space-between; /* align items along the main axis */
    align-items: center; /* align items along the cross axis */
}

6. Grid Layout

Grid properties:

.grid {
    display: grid;
    grid-template-columns: repeat(3, 1fr); /* 3 equal columns */
    gap: 10px; /* space between grid items */
}

7. Animations

Basic CSS animations:

@keyframes example {
    from {background-color: red;}
    to {background-color: yellow;}
}

.animated {
    animation-name: example;
    animation-duration: 4s;
}

8. Micro-interactions

Example of a hover effect:

.button {
    transition: background-color 0.3s ease;
}

.button:hover {
    background-color: blue;
    color: white;
}

9. Advanced Selectors

/* Child Selector */
.parent > .child {
    color: red;
}

/* Attribute Selector */
input[type="text"] {
    border: 1px solid #ccc;
}
