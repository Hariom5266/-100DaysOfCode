JavaScript Cheatsheet

1. Basic Syntax

// Single line comment
/* Multi-line comment */

let x = 5; // Declaring a variable
const y = 10; // Declaring a constant

2. Data Types

Common data types:

  • String
  • Number
  • Boolean
  • Object
  • Array
  • Undefined
  • Null

3. Functions

function myFunction(param) {
    return param * 2;
}

const myArrowFunction = (param) => param * 2;

4. Control Structures

// If statement
if (x > 5) {
    console.log("x is greater than 5");
} else {
    console.log("x is less than or equal to 5");
}

// Switch statement
switch (fruit) {
    case 'apple':
        console.log('Apple');
        break;
    default:
        console.log('Not an apple');
}

5. Loops

// For loop
for (let i = 0; i < 5; i++) {
    console.log(i);
}

// While loop
let j = 0;
while (j < 5) {
    console.log(j);
    j++;
}

6. Arrays

let fruits = ['apple', 'banana', 'orange'];
console.log(fruits[0]); // Accessing elements
fruits.push('grape'); // Adding elements

7. Objects

let person = {
    name: 'John',
    age: 30,
    greet: function() {
        console.log('Hello ' + this.name);
    }
};

person.greet(); // Calling method

8. ES6 Features

  • Template Literals: let message = `Hello, ${name}`;
  • Destructuring: const { name, age } = person;
  • Spread Operator: let newArray = [...fruits];
  • Promises: let promise = new Promise((resolve, reject) => { ... });

9. DOM Manipulation

const element = document.getElementById('myElement');
element.style.color = 'blue'; // Change style
element.innerHTML = 'Hello World!'; // Change content

10. Event Handling

element.addEventListener('click', function() {
    alert('Element clicked!');
});

11. Fetch API

fetch('https://api.example.com/data')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error('Error:', error));
